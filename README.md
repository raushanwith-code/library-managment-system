# 📚 Library Management System (JavaScript)

A simple and efficient **Library Management System** built using **JavaScript (Node.js)** with **JSON-based storage**.
This project demonstrates how real-world library operations can be implemented without using a database.

---

## 🚀 Features

* 📖 View all books
* ➕ Add new books
* ❌ Remove books
* 🔍 Search books by title or ID
* 📚 Issue books
* 🔄 Return books
* 💾 Data stored in JSON file (no database required)
* ⚡ Fast and lightweight CLI-based system

---

## 🛠️ Tech Stack

* **JavaScript (Node.js)**
* **JSON (File-based storage)**

---

## 📂 Project Structure

```
libraryManagement/
 ├── libraryMang.js   # Main application logic
 ├── data.json        # Book records (database)
```

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/raushanwith-code/library-managment-system.git
   ```

2. Navigate to the project folder:

   ```bash
   cd libraryManagament
   ```

3. Run the application:

   ```bash
   node libraryMang.js
   ```

---

## 💡 How It Works

* The system uses JavaScript functions to perform all operations
* Book data is stored and managed inside a JSON file
* Each operation (add, delete, issue, return) updates the JSON file
* No external database is required
